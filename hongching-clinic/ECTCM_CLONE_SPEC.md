## 每頁共用版面規格

### 統一 Layout
```
[breadcrumb: 配藥/收費 > 配藥/收費列表]
[BarCode/關鍵字: ___  掛號日期: 2026-03-29  搜索  當天記錄  刷新  進階查詢  快捷配藥/收費  結算鎖定  更多操作...]
[診所 | 掛號日期▲ | 排號 | 顧客編號 | 顧客姓名 | 性別 | 年齡 | 診治醫師 | 服務 | 再需付款 | 服務總款 | 配藥狀態 | 收費狀態 | 列印 | 操作]
[...data rows with zebra striping...]
[共計: XX 條記錄  1 2 3 4 5 6  跳轉到第__頁]
```

### 色彩規格
- Header/Logo bar: #006666 (dark teal)
- Nav bar: #5ba3a3 (medium teal)
- Sub-nav: #e8d4a0 (gold/beige)
- Table header: #008080 (teal)
- Filter bar: #ffffcc (light yellow)
- Zebra odd: #f5f5f5
- Zebra even: #ffffff
- Active nav: darker + underline
- Breadcrumb bg: #b8d4d4

### 字體
- 主要: 13px
- 表頭: 12px bold
- Breadcrumb: 12px
- 按鈕: 12px

### 按鈕樣式
- 搜索: dark green bg, white text
- 當天記錄: yellow/gold bg
- 刷新: dark blue bg
- 進階查詢: teal bg
- 快捷配藥/收費: yellow/gold bg
- 結算鎖定: dark grey bg
- 更多操作: green bg with dropdown arrow

## 核心頁面規格

### 1. 診症列表
- 表頭: 排號, 顧客編號, 顧客姓名, 性別, 年齡, 手機, 掛號日期, 掛號時間, 到診時間, 診治醫師, 現時狀態, 最後更新, 診所, 服務項目, 操作
- 快捷操作欄: 顧客編號/姓名/電話搜索 + 掛號日期 + 搜索顧客 + 新增顧客 + 掛號 + 快速掛號 + 醫師掛號表
- 搜索欄: 關鍵字 + 掛號日期 + 搜索 + 進階查詢 + 逾期未處理診症 + 顧客分析 + 病史備份
- 操作: 病歷(link)

### 2. 配藥/收費列表
- 表頭: 診所, 掛號日期▲, 排號, 顧客編號, 顧客姓名, 性別, 年齡, 診治醫師, 服務, 再需付款, 服務總款, 配藥狀態, 收費狀態, 列印, 操作
- 搜索欄: BarCode/關鍵字 + 掛號日期 + 搜索 + 當天記錄 + 刷新 + 進階查詢 + 快捷配藥/收費 + 結算鎖定 + 更多操作
- 配藥狀態: 配藥完畢✓ + 標為已取藥✓ + 提念(bell icon)
- 收費狀態: 已收費
- 列印: 處方A4/A5(中文/英文) + 配藥員處方 + 標籤機 + 詳情 + 收據
- 底部總計: 總數 | 再需付款合計 | 服務總款合計

### 3. 中藥管理
- 搜索欄: 關鍵字 + 搜索 + 刷新 + 導出Excel + 進階查詢 + 藥物分析 + 更多操作
- 表頭: 藥物資料 (展開式)
- 功能: 添加藥材/顆粒、批量操作、藥材供應商、庫存報告

### 4. 入貨管理
- 表頭: 系統編號▼, 診所名稱, 供應商名稱, 藥物總數, 類型, 更新庫存, 發票編號, 發票日期, 發票金額, 折扣率, 折扣金額, 折後金額, 建立人, 操作
- 搜索欄: 關鍵字 + 搜索 + 重置 + 刷新 + 導出Excel + 輸入貨單 + 進階查詢 + 入庫報表 + 按藥物查看入貨記錄
- 底部: 共計 + 折後金額合計

### 5. 顧客列表 (診所顧客列表)
- 表頭: 顧客編號, 顧客姓名, 性別, 年齡, 手機, 建檔日期, 求診次數, 主診醫師
- 快捷操作: 搜索顧客 + 新增顧客 + 掛號 + 快速掛號

### 6. 報表
- 三大類: 醫師工作報表、診所管理報表、醫務管理報表
- 每類有子報表（進階查詢 + PDF/Excel 導出）

## 實施優先順序

1. **導航結構** — 改成 eCTCM 一樣嘅 top nav + dropdown
2. **配藥/收費** — 最常用，每日必用
3. **診症列表** — 每日必用
4. **顧客列表** — 搜索病人
5. **中藥管理** — 庫存管理
6. **入貨管理** — 進貨
7. **報表** — 營運分析
